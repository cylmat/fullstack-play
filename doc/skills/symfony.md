
# Symfony

## Roadmap Symfony avancée

1. Dependency Injection   - Services   - Autowiring   - Autoconfiguration   - Service Container   - Paramètres
2. Event Dispatcher   - Events   - Event Subscribers   - Event Listeners   - Custom Events
3. Messenger   - Message Bus   - Handlers   - Middleware   - Async Workers   - Retry Strategy   - Dead Letter Queue
4. Security   - Authenticator   - Voters   - Roles   - Access Control   - JWT   - OAuth2   - Remember Me
5. Doctrine ORM   - Relations complexes   - DQL   - QueryBuilder   - Lazy Loading   - Eager Loading   - Lifecycle Callbacks   - Migrations   - Fixtures
6. Cache   - Cache Component   - Redis   - APCu   - Cache Pools   - HTTP Cache
7. API Platform   - Resources   - Serialization   - Filters   - Pagination   - GraphQL   - OpenAPI
8. Serializer   - Groups   - Context   - Normalizers   - Custom Normalizers
9. Validation   - Custom Constraints   - Validation Groups   - Callback Validation
10. Console    - Commandes personnalisées    - Arguments    - Options    - Progress Bar
11. Configuration    - YAML    - XML    - PHP Config    - Environment Variables
12. Twig    - Extensions    - Filtres personnalisés    - Fonctions personnalisées    - Components
13. Workflow    - State Machine    - Workflow    - Transitions
14. HTTP Client    - API REST    - Retry    - Async Requests
15. Scheduler    - Tâches planifiées    - Cron    - Jobs
16. Tests    - PHPUnit    - Tests fonctionnels    - WebTestCase    - Mocking    - Panther
17. Performances    - Profiler    - Debug Toolbar    - Blackfire    - Cache Doctrine
18. Logs    - Monolog    - Channels    - Handlers    - Rotation
19. Déploiement    - Docker    - Docker Compose    - CI/CD    - GitHub Actions    - Environment Management
20. Architecture    - DDD    - CQRS    - Hexagonal Architecture    - Repository Pattern    - Service Layer    - DTO    - Value Objects
