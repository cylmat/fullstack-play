# React

## Roadmap React avancée

1. Hooks avancés   - useReducer   - useMemo   - useCallback   - useRef   - useImperativeHandle   - useLayoutEffect   - useInsertionEffect   - useId   - Hooks personnalisés
2. Gestion de l'état   - Context API   - Reducers   - Zustand   - Redux Toolkit   - Jotai   - Recoil
3. Gestion des données serveur   - TanStack Query (React Query)   - SWR   - Cache   - Optimistic Updates   - Infinite Queries   - Pagination
4. Performance   - React.memo   - Memoization   - Lazy Loading   - Code Splitting   - Suspense   - Virtualisation de listes   - Profiler
5. Routing   - React Router   - Nested Routes   - Dynamic Routes   - Protected Routes   - Data Loaders
6. Formulaires   - React Hook Form   - Formik   - Validation (Zod, Yup)   - Formulaires dynamiques
7. Gestion des erreurs   - Error Boundaries   - Fallback UI   - Logging
8. Architecture   - Atomic Design   - Feature-Based Architecture   - Clean Architecture   - Separation of Concerns
9. TypeScript   - Génériques   - Utility Types   - Types inférés   - Custom Types   - Discriminated Unions
10. Tests    - Jest    - Vitest    - React Testing Library    - Mocking    - Tests d'intégration    - Snapshot Testing
11. Animations    - Framer Motion    - React Spring    - CSS Transitions
12. Accessibilité (a11y)    - Navigation clavier    - ARIA    - Focus Management
13. Optimisation    - Bundle Analysis    - Tree Shaking    - Lazy Imports    - Dynamic Imports
14. Communication    - WebSockets    - Server-Sent Events    - GraphQL    - REST API
15. Patterns avancés    - Compound Components    - Render Props    - Higher Order Components    - Controlled / Uncontrolled Components    - Portals
16. SSR / SSG    - Next.js    - Server Components    - Hydration    - Streaming
17. Sécurité    - XSS    - CSRF    - Sanitization    - Authentification JWT    - OAuth
18. Outils    - Vite    - ESLint    - Prettier    - Husky    - Storybook
